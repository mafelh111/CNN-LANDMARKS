Question: Outline the steps you took to get to your final CNN architecture and your reasoning at each step. Describe why you think the architecture is suitable for the current problem.

Answer:

Inicialmente se intentó entrenar una CNN desde cero con cuatro bloques convolucionales, pero el accuracy obtenido fue de apenas 45%, insuficiente para el objetivo del proyecto, lo que llevó a explorar transfer learning como alternativa. Se consideró primero ResNet-50 por su alto accuracy en ImageNet, pero resultó demasiado pesada para el hardware disponible no lograba cargar correctamente ni siquiera en Google Colab. Por eso se optó por ResNet-18, que es mucho más liviana (11 millones de parámetros) y aun así logró un accuracy final de 72% en el conjunto de prueba, superando ampliamente el objetivo del 50%.

¿Por qué es adecuada para el problema? En lugar de enseñarle al modelo desde cero qué es una textura, una forma o un patrón, se usa una red que ya aprendió todo eso entrenando con millones de imágenes. Solo se le enseña la parte nueva, que en este caso es distinguir entre los 50 landmarks específicos. Todos los parámetros de la red se congelaron excepto la última capa, que fue reemplazada por una nueva adaptada a las 50 clases.

Esto funciona especialmente bien para landmarks porque son estructuras con formas y patrones visuales muy distintivos, exactamente el tipo de características que ResNet ya sabe detectar.

from src.predictor import Predictor
from src.helpers import compute_mean_and_std

# First let's get the class names from our data loaders
class_names = data_loaders["train"].dataset.classes

# Then let's move the model_transfer to the CPU
# (we don't need GPU for inference)
model_transfer = model_transfer.cpu()
# Let's make sure we use the right weights by loading the
# best weights we have found during training
# NOTE: remember to use map_location='cpu' so the weights
# are loaded on the CPU (and not the GPU)
model_transfer.load_state_dict(
    torch.load("checkpoints/model_transfer.pt", map_location="cpu")
)

# Let's wrap our model using the predictor class
mean, std = compute_mean_and_std()
predictor = Predictor(model_transfer, class_names, mean, std).cpu()

# Export using torch.jit.script
scripted_predictor = torch.jit.script(predictor)
scripted_predictor.save("checkpoints/transfer_exported.pt")
